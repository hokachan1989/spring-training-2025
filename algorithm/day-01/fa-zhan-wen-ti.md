* 数値が0より大きいかどうかは `>` 演算子で確認できます

* 数値が0かどうかは `==` 演算子で確認できます

### 解答例

#### 1. 数値の正負判定

#### 4. BMI計算機

```python
# ユーザーから身長と体重を入力として受け取る
height_cm = float(input("身長(cm)を入力してください: "))
weight_kg = float(input("体重(kg)を入力してください: "))

# BMIを計算する
height_m = height_cm / 100  # cmからmに変換
bmi = weight_kg / (height_m ** 2)  # BMI計算式
print(f"あなたのBMIは{bmi:.2f}です。")

# BMIに基づいて健康状態を判定する
if bmi < 18.5:
    print("判定: 痩せ型")
elif bmi < 25:
    print("判定: 普通体重")
else:
    print("判定: 肥満")
```

#### 5. うるう年判定

```python
# ユーザーから年を入力として受け取る
year = int(input("西暦を入力してください: "))

# うるう年かどうかを判定する
if (year % 4 == 0 and year % 100 != 0) or (year % 400 == 0):
    print(f"{year}年はうるう年です。")

```

#### 6. 簡単な買い物計算

```python
# 購入金額を入力
amount = float(input("購入金額を入力してください: "))

# 会員カードの有無を入力（yesまたはno）
has_card = input("会員カードをお持ちですか？ (yes/no): ").lower() == "yes"

# 基本割引率を決定
if amount < 3000:
    discount_rate = 0  # 割引なし
elif amount < 5000:
    discount_rate = 0.05  # 5%割引
elif amount < 10000:
    discount_rate = 0.10  # 10%割引
else:
    discount_rate = 0.15  # 15%割引

# 会員カードの追加割引
if has_card:
    discount_rate += 0.02  # 追加2%割引

# 割引金額と最終的な支払金額を計算
discount_amount = amount * discount_rate
final_amount = amount - discount_amount

# 結果を表示
print(f"購入金額: {amount:.0f}円")
print(f"割引率　: {discount_rate * 100:.0f}%")
print(f"割引金額: {discount_amount:.0f}円")
print(f"支払金額: {final_amount:.0f}円")
```
